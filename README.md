# 日本全国指定緊急避難場所のオープンデータです。

国土地理院で公開している指定緊急避難場所データ（https://www.gsi.go.jp/bousaichiri/hinanbasho.html ）のCSVファイルをgeojson、jsonに変換し公開しています。長野市のみ独自のオープンデータ（http://linkdata.org/work/rdf1s6862i ）を加工し、取り込んでいます。

# 使い方
exampleを参照のこと。
